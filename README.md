# 🏆 SportsDeck — AI Sports News & Debate Arena

A real-time sports news hub and AI-powered debate partner built as a capstone project.

## 🌐 Live Demo
[https://sports-deck.vercel.app](https://sports-deck.vercel.app)

## 📋 Project Overview
SportsDeck is a full-stack web application that combines daily sports news with an AI debate chatbot called CoachBot. Users can read today's top sports stories across NBA, NFL, MLB, NHL, Soccer, UFC, and Boxing, then jump into the Debate Arena to argue any sports topic with an AI analyst that uses real-time web search.

## ✨ Features
- Daily sports news feed updated with real-time AI search
- Filter news by NBA, NFL, MLB, NHL, Soccer, UFC, Boxing
- Debate Arena with full multi-turn chat
- Choose to argue FOR or AGAINST any topic
- CoachBot searches the web for current scores and stats
- All debates saved to Supabase database
- Click any news story to debate it instantly
